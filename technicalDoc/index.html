What is Python?

Python is an interpreted, object-oriented, high-level programming language. Python's simple, easy to learn syntax emphasizes readability. Python supports modules and packages, which encourages program modularity and code reuse. The Python interpreter and the extensive standard library are available in source or binary form without charge for all major platforms, and can be freely distributed.

Like JavaScript (and unlike Java), Python supports a programming style that uses simple functions and variables without engaging in class definitions. However, for JavaScript, that's all there is. Python, on the other hand, supports writing much larger programs and better code reuse through a true object-oriented programming style, where classes and inheritance play an important role.

Data Types

Python has eight data type categories:

  • Text
  • Numeric
  • Boolean
  • Sequence
  • Mapping
  • Set
  • Binary
  • None

Each of these data types are built-in by default and are assigned to variables. You can determine the data type of a variable by using the type() function. You can also set a specific type by using a constructor function, which is they type of data type you want to use followed by parentheses. For example:

  • x = range(6)
  • x = bytes(5)
  • x = list(("apple", "banana", "cherry"))
Conditionals

Conditionals are used when you need a portion of code to run under certain conditions. This allows your code to run more smoothly, and keeps code from running unnecessarily, which can slow it down. Conditionals can be used in many ways, but are often used with Logical and Relational Operators, If...Else Statements, and Loops.

Relational Operators are borrowed from mathematics, and are used to compare the relationships of two values. Relational operators belong to an even larger family called boolean operators, which are operators which return a boolean. There are six different relational operators:

  • equal to (==)
  • not equal to (!=)
  • greater than (>)
  • less than (<)
  • greater than or equal to (>=)
  • less than or equal to (<=)
print(1 == 1) # True print("Steve" == "Bob") # False print(10 > 9) # True

Another way to run code conditionally is by using If Statements. Code following an If Statement only runs if the conditional statement is True. x = 10 if x == 10: print("x is equal to 10")

Suppose that our condition in an if statement doesn't work. The code doesn't run, and then the program just runs whatever else afterwards. What if we want something special to happen if that condition wasn't True? This is when you would add an Else Statement.

An else statement is a backup piece of code that you want to run if the condition in an if statement results to False. You can add an else statement to an if statement. The code in the else statement runs when the expression in the if is False.

time = 3 if time > 8 and time < 20: print("Hi! What would you like to order?") else: print("Why are you here when the parlor's closed? Go home.")

if and else statements are nice, but we still need an additional level of control over decisions sometimes. The elif statement helps by introducing fallback conditions that we can check for if the if statement condition fails, but before the else block is executed. For example:

x = 20 if x < 10: print("x is less than 10") elif x < 30: print("x is less than 30") # This will print else: print("x is greater than or equal to 30")
Loops

Python has two primative loop commands:

  • while loops
  • for loops

While Loops will help us repeat certain lines of code without having to type repeated lines over and over again. This will help us cut down our code and make things more efficient. But first, let's go over the structure and functionality of a while loop...

The while loop loops through a block of code as long as a supplied boolean condition is True. Often times, you need to have your condition be able to be broken due to something in the while loop.

i = 1 while i < 6: print(i) i += 1

For loops are loops that run for a predefined amount of time that can't be altered. They are simpler to write than a while loop and do not run the risk of potentially running forever, which can be rather easy to accidentally do depending on your program. We can use for loops like while loops, as well. for loops are better to use when you want to run code a certain number of times.

for i in range(5): print("This loop runs 5 times")
Functions

A function is a block of code which only runs when it is called. To call a function, use the function name followed by parenthesis. You can pass data, known as parameters, into a function. Information can be passed into functions as arguments. Arguments are specified after the function name, inside the parentheses. You can add as many arguments as you want, just separate them with a comma. A function can return data as a result.

def my_function(): print("Hello from a function")

Python also accepts function recursion, which means a defined function can call itself. Recursion is a common mathematical and programming concept. It means that a function calls itself. This has the benefit of meaning that you can loop through data to reach a result. The developer should be very careful with recursion as it can be quite easy to slip into writing a function which never terminates, or one that uses excess amounts of memory or processor power. However, when written correctly recursion can be a very efficient and mathematically-elegant approach to programming.

def tri_recursion(k): if(k > 0): result = k + tri_recursion(k - 1) print(result) else: result = 0 return result print("\n\nRecursion Example Results") tri_recursion(6)
